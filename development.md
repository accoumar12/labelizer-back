# Development

## Run the project

```bash
docker compose up
```

Be careful, work with the frontend on the develop branch, because the main branch lags behind...

## Getting started

### Run the tests

```bash
pytest
```

### Play with the sample data

1. Launch the app.
2. Go to `http://localhost:5173/db`.
3. Upload sample data located in `data_example/test_data.zip`, in the **Upload Database** section.
4. Play with the data in the **Labelling** and **Validation** tabs!
